WPF-приложение для мониторинга аппаратного обеспечения компьютера в реальном времени

Описание
HardwareMonitor — настольное приложение для комплексного мониторинга аппаратных ресурсов компьютера.
Программа получает данные о процессоре, оперативной памяти, видеокарте, дисках, сетевых адаптерах и системных параметрах с помощью технологии WMI (Windows Management Instrumentation) и отображает их в удобном интерфейсе WPF.
Приложение реализовано с использованием архитектурного паттерна MVVM и асинхронных операций для обеспечения высокой производительности и отзывчивости интерфейса.

Цель лабораторной работы
Разработать WPF-приложение для мониторинга аппаратного обеспечения компьютера с использованием технологии WMI и архитектурного паттерна MVVM.

Задачи
  -Освоить работу с WMI в C# через библиотеку System.Management
  -Получать информацию об аппаратных компонентах системы
  -Реализовать асинхронное обновление данных без блокировки интерфейса
  -Использовать архитектурный паттерн MVVM
  -Организовать параллельное выполнение запросов к оборудованию
  -Реализовать экспорт данных системы

Функциональность
  Процессор (CPU)
    -Название и производитель
    -Количество ядер и потоков
    -Архитектура и частота
    -Текущая загрузка в реальном времени
  Оперативная память (RAM)
    -Общий и доступный объём
    -Процент использования
    -Информация о модулях памяти
  Видеокарта (GPU)
    -Название устройства
    -Объём видеопамяти
    -Текущая загрузка
  Диски
    -Физические диски (модель, размер, тип)
    -Логические разделы (буква, объём, файловая система, свободное место)
  Сетевые адаптеры
  Название адаптера
    -MAC-адрес
    -Состояние подключения

Структура проекта
HardwareMonitor/
├── Models/ — модели данных оборудования
├── Services/ — сервисы получения данных через WMI
├── ViewModels/ — логика MVVM
├── Views/ — интерфейс приложения
├── Utils/ — экспорт и вспомогательные функции
└── App.xaml — точка входа приложения

Как использовать
  -Запустите приложение
  -Данные загружаются автоматически при старте
  -Переключайтесь между вкладками оборудования
  -Используйте кнопку обновления для ручного обновления
  -На вкладке процессов можно завершать активные задачи
  -Кнопка экспорта сохраняет отчёт о системе
